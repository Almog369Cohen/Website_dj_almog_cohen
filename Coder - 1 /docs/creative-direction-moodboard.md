# Creative Direction & Moodboard v1

- **Brand palette**
  - Blue: #059cc0
  - Green: #03b28c
  - Dark Gray: #1f1f21
  - White: #ffffff
  - Gradient: linear-gradient(135deg, #059cc0 0%, #03b28c 100%)

- **Typography (Hebrew-first)**
  - Headings: Assistant / Almoni Neue (700–800)
  - Body: Noto Sans Hebrew / Assistant (400–500)
  - Line-height: 1.2 (H1–H2), 1.5 (body)
  - Fallbacks: system-ui, Arial

- **Visual style**
  - Modern, minimal, dynamic. Dark elegant background with blue/green glow.
  - Soundwave motion motifs, light beams, subtle gradients.
  - Clean layout, strong hierarchy, generous spacing.

- **Photo treatment**
  - Base: Black-and-white performance and crowd photos.
  - Overlay: Blue/green gradient at 40–60% using Screen/Overlay or soft vignette.
  - Focus on emotion: smiles, hands up, chuppah/first dance moments, students.

- **Motion & micro-interactions**
  - Hero: 8–12s muted, looped live performance video; oflow glow edges.
  - CTA hover: soft neon outline + glow.
  - Background: faint animated gradient or soundwave lines at 2–4% opacity.

- **Component style**
  - Buttons: pill or rounded with gradient stroke + dark fill.
  - Cards: dark glass with subtle inner border (1px) and soft shadow.
  - Sections: high contrast; headings with subtle glow/shadow.

- **Copy tone**
  - Confident, human, warm, premium. Clear CTAs.

- **Hero headline (Hebrew suggestion)**
  - "אלמוג כהן – יוצר רגעים של אנרגיה, רגש וחיבור אנושי דרך הסאונד."
  - CTAs: "שריין תאריך לאירוע" / "הצטרף לקורס הדיג'יי"

- **Instagram inspiration (from @dj_almog_cohen)**
  - Pull 6–9 tiles: 3 live crowd moments, 2 student/studio shots, 1–2 behind-the-scenes.
  - Suggested highlights: "מאחורי הקלעים", "אימונים", "אירועים", "תלמידים".
  - Embed approach for website MVP: Instagram embed widget; upgrade later to Graph API.

- **Accessibility**
  - RTL-first. Target contrast ratio ≥ 4.5:1.
  - Min body size 16px; tap targets ≥ 44px.

- **Approval checklist**
  - Palette/gradient OK?
  - Primary font choice (Assistant vs Noto Sans Hebrew)?
  - Photo treatment direction OK?
  - Motion intensity (subtle vs bold)?
